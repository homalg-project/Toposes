4.1 Test functions

4.1-1 BraidedCartesianCategoriesTest
‣ BraidedCartesianCategoriesTest( cat, a, b )( function )

The arguments are

  • a CAP category cat

  • objects a, b

This function checks for every operation declared in BraidedCartesianCategories.gd if it is computable in the CAP category cat. If yes, then the operation is executed with the parameters given above and compared to the equivalent computation in the opposite category of cat. Pass the option 'verbose := true' to output more information.
